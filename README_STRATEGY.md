# MAISON VALESKA - Haute Joaillerie

## 1. Branding & Naming
- **Nombre**: Maison Valeska
- **Significado**: Valeska (Eslavo: "Líder Gloriosa"). Evoca herencia, poder y refinamiento.
- **Posicionamiento**: Joyería de inversión y piezas únicas.
- **Eslogan**: *L'Art de l'Éternité* (El Arte de la Eternidad).

## 2. Estrategia de Diseño
- **Paleta**: #0A0A0A (Black Ink), #FFFFFF (Pure), #C5A059 (Champagne Gold), #F4F1EA (Silk).
- **Tipografía**: Playfair Display (Serif) para títulos, Inter (Sans) para legibilidad.
- **UX**: Navegación "Ghost", transiciones de página suaves, micro-interacciones en botones de compra.

## 3. Arquitectura Técnica
- **Frontend**: React 19 + Tailwind CSS + Framer Motion.
- **Backend**: Node.js (Express) + SQLite (Better-SQLite3).
- **SEO**: Estructura semántica, JSON-LD para productos, Meta tags dinámicos.
- **Pagos**: Integración con Stripe (Simulada).

## 4. Roadmap de Desarrollo
1. Setup de Servidor y Base de Datos.
2. Implementación de UI Core (Navbar, Hero, Footer).
3. Catálogo de Productos y Vista de Detalle.
4. Sistema de Carrito y Checkout.
5. Panel de Administración (Básico).
